# Research Papers Repository

## Paper Index Maintenance
Whenever a research paper is added or removed from this repository, **always update `INDEX.md`** at the repo root:
- Add new papers to the correct thematic group with folder link and one-line summary
- Remove entries for deleted papers
- If a paper doesn't fit an existing group, create a new group or place it in the closest match

## Thematic Groups
Papers are organized into five groups in INDEX.md:
1. **Human Behavior Simulation & Evaluation** — LLMs mimicking real humans
2. **LLM Agent Systems & Domain Simulations** — Multi-agent systems for specific domains
3. **A/B Testing with LLM Agents** — LLM agents replacing real-user experiments
4. **RL for LLM Post-Training** — Reinforcement learning methods for training LLMs
5. **Architecture, Efficiency & Scaling** — Model architecture, inference, context, and scaling

## Repo Structure
Each paper lives in its own folder with:
- `paper.pdf` — the original PDF
- `REPORT.md` — comprehensive analysis report
- `figures/` — extracted key figures
